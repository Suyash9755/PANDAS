This pandas repository covers multiple projects with pandas 

Skills & Tools Covered:
Data Manipulation: Using Pandas DataFrames for indexing, slicing, and cleaning.

Statistical Analysis: Implementing Karl Pearson’s Correlation and descriptive statistics (Mean, Sum, Variance).

Data Visualization: Plotting relationships using Matplotlib and Seaborn.

Mathematical Implementation: Translating statistical formulas into functional Python code.

Feature Engineering: Understanding how variables relate to one another for predictive modeling.
